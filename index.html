How to Contribute

Fork this repository.

Create a new folder named after your project in

/winnipeg-programs-directory

Add your source code and a README.md in your project folder that includes:

  • Project name
  • Short description
  • How to run it
  • Author info (name, link, etc.)

Commit your changes to your fork.

Open a Pull Request to the main branch of this repo.

Your project will be reviewed and merged for everyone to see!
